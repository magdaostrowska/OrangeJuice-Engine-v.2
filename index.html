About Orange Juice Engine

Orange Juice Engine is the result of the semester long assingment for the 3D Engines subject in the Video Game Design and Development Degree at CiTM, UPC. This assignment has been done under the mentory of Rub√©n Ajenjo Rodriguez, teacher of the subject.

Orange Juice Engine Features:

  • Mesh and texture loading, from within the engine and through drag and drop.
  • Scene serialization.
  • Mousepicking within scene, as well as object hirearchy and transformation from within the editor.
  • Particle emmiter creation and editing, which was the specialisation for our engine.

Both the project files and releases are accessible in our repository.

Sub-systems

  • Particle System

  • Sub-Systems

    • Particle System

    For our last assignment, where we had to choose a specialization sub-system for our engine, we chose to make Orange Juice Engine a Particle Specialized engine.

    Our particle system features:

    • Creating and editing particle emitters as objects.
    • Editing particle contents from editor, such as color, shape and texture.
    • Editing emitter speed, shape and particle emision frequency.

    Our aim with this particle system is to create a simple way in which to create multiple particles which accomplish different effects, through a combination of different variables, so as to make it both reliable, simple and flexible.

    Initial phase of emitter development.

    Emitters being used to display smoke coming from chimneys.

    Other Sub-Systems

    Orange Juice Engine features multiple other sub-systems which allow functioning much similar to that of a regular engine.

    • Scene saving and loading from within the engine.
    • Mesh loading and drag and dropping from file folder.
    • Primitive generation from within the engine.
    • Texture loading and drag and dropping from file folder.
    • Object hirearchy and inheritance from parents.
    • Object manipulation, such as transformations, orientation changes or size changes within editor, as well as mousepicking within scene.
